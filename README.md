# 灯火の街リーザ

スマートフォン専用のWEBブラウザゲーム(街づくり×放置クリッカー)です。
HTML / CSS / JavaScript のみで動作し、ビルド・サーバー・外部通信は一切不要です。

## 体験の核

「ちいさな灯をまもる約束が、いつのまにか、たくさんの人の笑顔になっていた」

対象は子供から大人まで。文章は小学3年生が読める水準(難しい漢字にはふりがな)で、大人が読んでも美しい言葉を選んでいます。

## ゲーム概要

むかし、ここにはなにもなかった。あったのは、たいせつな友だちがのこしてくれた、ちいさなランタンひとつだけ。「この灯を、いつか街いっぱいの光にしてね」——その約束のために、レインたちと一緒に荒れ地へ街を育てていく物語です。

資源を集め、施設を建て、住人が増え、街が少しずつ明るく、にぎやかになっていく過程そのものがこのゲームの体験の核です。

## 遊び方

1. はじめて起動すると、タップで進む**オープニング「約束」**(4枚)が流れます(右上でスキップ可。「きろく」からいつでも再生できます)
2. **画面上部の「街ビュー」をタップ**すると「✦ ひかり」が集まります(光の粒が舞い、街の灯が脈動します)
3. ひかりを使って**施設をたてる・そだてる**。施設は資源をじどうで集め、**街ビューに対応する建物が実際に出現・成長**します
4. 市場は「▤ もくざい」を集めます。上位施設に必要です
5. 施設が育つと**🏠 住人**が増えます(合計Lv×2人)。住人が増えると街の窓明かりも増え、節目では名前つきの住人が引っ越してきます
6. 施設の合計レベルで「**街のかがやき**」が上がり、しきい値をこえると**発展段階**が進みます。お祝いの演出とストーリーが解放されます
7. 迷ったら資源バー下の「**次の目標**」チップを見れば、次の一手が分かります

### 資源

| 資源 | 集めかた | つかいみち |
|---|---|---|
| ✦ ひかり | タップ/ランタン工房 | すべての施設 |
| ▤ もくざい | 市場 | 上位施設 |
| 🏠 住人 | 施設の合計Lv×2人 | 街のにぎわい(見える指標) |

### 施設(5種類)

| 施設 | なかま | はたらき | はじめのコスト |
|---|---|---|---|
| ランタン工房 | レイン | ✦がじどうで集まる(+1/秒×Lv) | ✦15 |
| 市場 | バルト | ▤がじどうで集まる(+1/2秒×Lv) | ✦40 |
| 学問所 | アルノ | タップで集まる✦がふえる(+1×Lv) | ✦100・▤10 |
| 救護院 | リーザのねがい | じどうで集まる量+10%×Lv | ✦250・▤30 |
| 街灯網 | ユリウス | かがやき+5×Lv・街の光がふえる | ✦600・▤80 |

コストはLvごとに約1.6倍ずつ増えます。

### 発展段階(5段階)

| 段階 | 名称 | 必要かがやき |
|---|---|---|
| 1 | 泥のアジト | 0 |
| 2 | 開拓の集落 | 5 |
| 3 | 灯りの街 | 14 |
| 4 | 交易都市 | 30 |
| 5 | 光の都リーザ | 55 |

段階が上がる瞬間は、光の波と「おめでとう!」の祝祭演出(光の粒+住人たちの歓声)が流れ、ストーリーが解放されます。

## スマホ操作方法

- **街ビュー(上部)をタップ** … ひかりを集める(連打OK)
- **施設カードの「たてる/そだてる」** … 資源を使って施設をレベルアップ
- **トースト(画面下の吹き出し)** … 仲間の一言や住人のお知らせ。タップで閉じられます
- **フッターメニュー**
  - 「街」「しせつ」… メイン画面
  - 「きろく」… オープニングの再生と、解放済みストーリーの再読
  - 「せってい」… セーブ情報の確認・リセット
- 縦持ち・片手・親指操作前提(横スクロールなし、ボタンは44px以上)

## GitHub Pages公開手順

1. このリポジトリの **Settings → Pages** を開く
2. **Source** で「Deploy from a branch」を選択
3. **Branch** で `main` / `/ (root)` を選択して **Save**
4. 数分待つと `https://<ユーザー名>.github.io/novel8/` で公開されます

※ ローカルで試す場合は `index.html` をブラウザで開くだけで動作します。

## 実装済み機能

- オープニング「約束」(タップで進む4枚、スキップ可、「きろく」から再生可)
- タップによるひかり獲得(光の粒・街の灯の脈動・カウントアップ表示)
- 施設5種の「たてる/そだてる」と、街ビューへの建物の出現・成長(セーブ復元でも同じ街並みを再現)
- 住人システム(合計Lv×2人。窓明かりが住人数と連動。節目では職業+一言つきの住人10人)
- 仲間の声(レイン=青/ランタン、バルト=橙/天秤、アルノ=白銀/羽根ペンの吹き出しトースト。各5種以上)
- 「はじめての建設」「はじめての住人」では必ず仲間が声をかける
- 発展段階5段階(段階=土地と空気、施設=建物、と役割分離)と光の波+「おめでとう!」祝祭演出
- 段階到達ストーリー(原文のまま・ふりがな付き)と「きろく」からの再読
- 「次の目標」チップ(状況から自動判定)
- 子供が読めることば(ひかり/もくざい/住人/かがやき/たてる/そだてる、ふりがな付き)
- localStorage自動セーブ(5秒ごと+操作時+画面非表示時)・確認つきリセット
- 数値の日本語略記(1.2万 など)、スマホ縦画面(390px基準)、蜂蜜色〜こはく色の暖色UI

## localStorage保存内容

キー: `toukaLizaSave_v1`

```json
{
  "v": 1,
  "maso": 123.4,
  "shizai": 56.7,
  "lv": { "lantern": 3, "market": 2, "school": 1, "clinic": 0, "lights": 0 },
  "unlocked": [2, 3],
  "maxStage": 3,
  "introSeen": true,
  "lastResidents": 12,
  "lastSaved": 1750000000000
}
```

| 項目 | 内容 |
|---|---|
| `maso` / `shizai` | 所持資源(表示名は「ひかり」「もくざい」。内部名は互換性のため維持) |
| `lv` | 各施設のレベル(街並み・住人数はここから決定的に再計算) |
| `unlocked` | 解放済みストーリーの段階番号 |
| `maxStage` | 到達済みの最高発展段階 |
| `introSeen` | オープニング「約束」を見たか |
| `lastResidents` | 住人トースト用の前回住人数 |
| `lastSaved` | 最終保存時刻(UNIXミリ秒) |

## セーブリセット方法

1. フッターメニューの「**せってい**」をタップ
2. 「**セーブデータを リセット**」をタップ
3. 確認ダイアログで「OK」を選ぶと、最初から始まります

(手動で消す場合はブラウザの開発者ツールから localStorage の `toukaLizaSave_v1` を削除してください)

## 既知の課題

- バックグラウンドタブでは生産が最大300秒ぶんまでしか加算されません(暴走防止のための仕様)
- localStorageが使えない環境(プライベートブラウズ等)ではセーブされません(プレイ自体は可能)
- 横画面・タブレットでも動作はしますが、レイアウトは縦持ちスマホに最適化されています
- 効果音・BGMは仕様により未実装です
- 仲間の紋章(🕯/⚖/✒)は端末の絵文字フォントに依存します(非常に古い端末では表示が変わる場合があります)
