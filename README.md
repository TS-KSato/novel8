# 灯火の街リーザ

スマートフォン専用のWEBブラウザゲーム(街づくり×放置クリッカー)です。
HTML / CSS / JavaScript のみで動作し、ビルド・サーバー・外部通信は一切不要です。
3D描画には同梱のThree.js(MITライセンス)を使用します(外部CDN読み込みなし)。

## 体験の核

「ちいさな灯をまもる約束が、いつのまにか、たくさんの人の笑顔になっていた」

対象は小学校高学年〜高校生以上。通常の漢字仮名交じり文で、難読語(救護院・街灯網・帳など)にのみふりがなを振っています。

## ゲーム概要

昔、ここには何もなかった。あったのは、大切な友人が遺してくれた、小さなランタンひとつだけ。「この灯を、いつか街いっぱいの光にしてね」——その約束のために、レインたちと一緒に荒れ地へ街を育てていく物語です。

資源を集め、施設を建て、住人が増え、街が少しずつ明るく、にぎやかになっていく過程そのものがこのゲームの体験の核です。

## 遊び方

1. 初めて起動すると、タップで進む**オープニング「約束」**(4枚)が流れます(右上でスキップ可。「図鑑」からいつでも再生できます)
2. **画面上部の「街ビュー」をタップ**すると「✦ 光」が集まります(光の粒がランタンから舞い上がります。夜は少し華やぎます)
3. 光を使って**施設を建てる・育てる**。施設は資源を自動で集め、**3Dの街に対応する建物が実際に出現・成長**します
4. 市場は「▤ 木材」を集めます。上位施設に必要です
5. 施設が育つと**🏠 住人**が増えます(合計Lv×2人)。住人が増えると街の窓明かりも増え、節目では名前つきの住人が引っ越してきます
6. 施設の合計レベルで「**街の輝き**」が上がり、しきい値を超えると**発展段階**が進みます。祝祭の演出とストーリーが解放されます
7. 街には**朝→昼→夕→夜**の時間が流れます(1周約4分)。夕方になると窓に一つずつ灯が入り、夜は灯りがいちばん輝きます
8. ときどき空に**願い星**(夜=流れ星/昼=光る蝶)が現れます。約6秒で消える前にタップすると、光・木材をまとめて獲得(自動生産45秒ぶん、序盤は15)
9. 迷ったら資源バー下の「**次の目標**」チップへ。2種類以上建てられるときは「次はどれを育てる?」とあなたの選択に委ねられます
10. 街ビュー右上の**掲示板**から「街のおねがい」を受けられます(下記参照)

### 資源

| 資源 | 集めかた | つかいみち |
|---|---|---|
| ✦ 光 | タップ/ランタン工房 | すべての施設 |
| ▤ 木材 | 市場 | 上位施設 |
| 🏠 住人 | 施設の合計Lv×2人 | 街の賑わい(見える指標) |

### 施設(5種類)

| 施設 | 担当 | 効果 | 初期コスト |
|---|---|---|---|
| ランタン工房 | レイン | ✦が自動で集まる(+1/秒×Lv) | ✦15 |
| 市場 | バルト | ▤が自動で集まる(+1/2秒×Lv) | ✦25 |
| 学問所 | アルノ | タップで集まる✦が増える(+1×Lv) | ✦70・▤6 |
| 救護院 | リーザの願い | 自動生産+10%×Lv | ✦250・▤30 |
| 街灯網 | ユリウス | 輝き+5×Lv・街の光が増える | ✦600・▤80 |

コストはLvごとに約1.6倍ずつ増えます。

### 発展段階(5段階)

| 段階 | 名称 | 必要な輝き |
|---|---|---|
| 1 | 泥のアジト | 0 |
| 2 | 開拓の集落 | 5 |
| 3 | 灯りの街 | 14 |
| 4 | 交易都市 | 30 |
| 5 | 光の都リーザ | 55 |

段階が上がる瞬間は、光の波と「おめでとう!」の祝祭演出(光の粒+住人たちの歓声)が流れ、ストーリーが解放されます。

## スマホ操作方法

- **街ビュー(上部)をタップ** … 光を集める(連打OK)。施設をタップすると対応カードへ移動
- **施設カードの「建てる/育てる」** … 資源を使って施設をレベルアップ
- **トースト(画面下の吹き出し)** … 仲間の一言や住人のお知らせ。タップで閉じられます
- **フッターメニュー**
  - 「街」「施設」… メイン画面
  - 「図鑑」… オープニングの再生/街のあゆみ(ストーリー再読)/ものがたり(条件達成で1ページずつ解放される8ページの読み物)
  - 「設定」… セーブ情報の確認・リセット
- 縦持ち・片手・親指操作前提(横スクロールなし、ボタンは44px以上)

## GitHub Pages公開手順

1. このリポジトリの **Settings → Pages** を開く
2. **Source** で「Deploy from a branch」を選択
3. **Branch** で `main` / `/ (root)` を選択して **Save**
4. 数分待つと `https://<ユーザー名>.github.io/novel8/` で公開されます

※ ローカルで試す場合は `index.html` をブラウザで開くだけで動作します。

## キャラ画像（顔アイコン・立ち絵）

会話の吹き出しと図鑑にキャラ画像を使用します。

- **帰属**: キャラ画像（顔 `*_face.png`・立ち絵 `*_full.png`、`assets/portraits/`）はプロジェクトオーナー提供。**商用利用可否は未確認のため保留**。確認が取れ次第更新します
- **吹き出し**: 仲間の一言・依頼主表示に、対応キャラの顔アイコン（円形クロップ・キャラ色の縁取り）を左、台詞を右に並べて表示。**画像読み込み失敗時は紋章（🕯/⚖/✒）にフォールバック**
- **図鑑**: 「ものがたり」のレイン/バルト/アルノのページに立ち絵（768×1024）を表示。未解放は「？」シルエット、解放でふわっと出現。**読み込み失敗時はテキストのみで正常表示**
- 顔は起動時にプリロード（チラつき防止）。立ち絵は図鑑を開いた時に読み込み
- 立ち絵は表示品質を保ったままパレットPNG化で軽量化（rein 93KB / bart 70KB / arno 63KB、768×1024・透過は維持）
- ※ 3Dの街にはキャラ画像を配置していません（風景の純度を保つため）

## キャラクター（レイン）

ランタン工房の前に、主人公レインの3Dモデル（`assets/models/rein.glb`）が常駐します。

- **帰属**: キャラモデルはプロジェクトオーナー提供。**商用利用可否は未確認のため保留**（モデルのメタデータ上の生成元は Tripo）。確認が取れ次第ここを更新します
- マテリアルは街と同じ MeshLambertMaterial に寄せ（写実的なテカリ・陰影を抑制）、街と同じ太陽・半球光だけで陰影を簡素化。昼夜サイクルの色が街と一緒にレインにも乗ります
- テクスチャは 2048→1024 に縮小。接地影を1つ（街の建物と同質のブロブ）。腕は検証済みの左右対称な「腕下げ」ポーズを最初から適用
- 待機中はゆっくり呼吸（上下動）、ランタン工房を育てる／レインをタップで小さくジャンプ
- **読み込み失敗・WebGL失敗・GLTFLoader未読込のいずれでも、レインは出ずに街は完全に正常表示**されます（フォールバックはテストで担保）

## 3D街ビュー(Three.js)

街ビューは本物の3Dのローポリ・ミニチュアジオラマです。

- **同梱ライブラリ**: Three.js r128 (`vendor/three.min.js`)。MITライセンスで、全文を `vendor/THREE-LICENSE.txt` に同梱。CDN読み込みは行いません
- **全ジオメトリをコード生成**(Box/Cylinder/Cone/Sphere等の組み合わせ)。外部の3Dモデル・テクスチャ・画像は不使用
- **アイソメ風カメラ**: OrthographicCamera(平行投影)による俯瞰35度・方位45度のダイヤモンド構図。台座の側面が見え、建物の正面壁が主役。角度・ズームは定数(`CAM`)で調整可能
- **配色の骨格**(全色は`town-model.js`の`PALETTE`定数に一元化):
  - ベース(画面の50-60%): しっかりした明るい草色の地面(#93c161)+クリームベージュの土(#e3cfa4)
  - セカンダリ(25-35%): 白壁(#f7efdc系)×はっきりした識別色の屋根(工房=レイン青 #4f86c9/市場=バルト橙 #e8893c/学問所=アルノ紺 #3c5188/救護院=緑 #7eb069/民家=テラコッタ・マスタード等)
  - アクセント(5-10%): 茶の小物(縁石 #8a6240・樽・看板・幹)が画面を引き締める
  - 道=一段明るいクリーム石畳(#eadfbf)+茶の縁石、台座側面=チョコ茶(#6f4f33)
- **ライティング**: ACESFilmicトーンマッピング+sRGB出力。HemisphereLight(空=薄青白/地=暖ベージュ)を主光源、DirectionalLightは補助+影用。朝夕も緑が濁らない白寄りの半球光
- **空**: GLキャンバスは透過し、CSSの上下グラデーション(昼=空色→白み/夕=桃→琥珀/夜=藍→青)+流れる雲+星をそのまま生かす
- **影のチラつき対策**: 太陽角度はゲーム内30秒ごとの離散更新(2秒で滑らかに補間)。シャドウマップ1024px・盤面にタイトなfrustum・bias/normalBias設定・細いパーツはcastShadow無効・夜は太陽影を無効化
- 島型ボード(台座の厚みあり)に広場・十字の道・丘状の緑地・ローポリツリー(幹+葉、最大12本)。段階に応じて石畳・花壇・水路が増える
- 施設は屋根形状・色・付属物(煙突/日よけ/塔/旗/街灯ポール)で識別でき、各建物5パーツ以上・Lvで4段階に増築。民家は住人数連動(上限14軒)
- 生命感: 煙突の煙、旗のはためき、夕暮れに窓へ一つずつ入る灯、3D空間に現れる願い星(流れ星/光る蝶)
- フィギュア感: 屋根の軒の出と棟の厚み、白い窓枠、立体のドア
- **施設をタップすると、対応する施設カードへスクロール&ハイライト**(Raycaster)
- 性能: ピクセル比上限1.5、共有ジオメトリ・マテリアル、transformのみのアニメーション
- 接地影: 台座の真下に薄い楕円の影を描き「置物」感を出す

### フォールバック

WebGL非対応・Three.js読み込み失敗・初期化失敗時は、自動的にCSSジオラマ(town3d.js)へ切り替わります。さらに `window.LIZA_USE_DIORAMA = false` で旧・横から見た夜景描画にも戻せます。手動制御は `window.LIZA_USE_3D = false`(3D無効化)。

## 街のおねがい(依頼システム)

住人や仲間から「おねがい」が届きます(街ビュー右上の掲示板に通知バッジ。同時最大2件)。

- **納品型**(5種): 「木材を80届けてほしい」など。達成すると資源+**装飾**(花壇・ベンチ・祝祭の旗・噴水・光の像)が3Dの街に実際に置かれ、セーブで永続します
- **達成型**(4種): 「学問所をLv5にしてほしい」など。大きめの資源報酬+依頼主の一言
- **二択型**(2種): 「祭りの準備、どちらを手伝う?」。どちらを選んでも正解ですが効果が異なり、選んだ側に10分間の生産ブースト(+15〜25%)が付きます

依頼は街の発展段階に応じて出現します。依頼をこなすほど、街が目に見えて豊かになります。

## CSSジオラマ(フォールバック)の構造

WebGLが使えない環境では、CSSのみの擬似俯瞰ミニチュアマップに自動で切り替わります(画像・外部ライブラリ不使用)。

- `#diorama`(perspective) の中に、`rotateX` で奥に傾けた地面ボード `#town-board` を置く
- 盤面の中央に「広場」と「**はじまりのランタン**」(常時灯る・このゲームの象徴)。広場から十字に道が伸びる
- 各施設は固定区画: ランタン工房=北 / 市場=東 / 学問所=西の丘 / 救護院=南 / 街灯網=道沿いのポール
- 建物は「立て看板(スタンディ)」方式: 盤面の上で逆方向に `rotateX` して立ち上げ、正面+側面+屋根のdiv構成でミニチュアの立体感を出す
- **施設Lvで建物が4段階に成長**: Lv1-2=小屋 → Lv3-5=看板つき → Lv6-9=2階+旗 → Lv10+=特別装飾
- **住人4人ごとに民家が1軒**増える(最大14軒。超過ぶんは地面の灯の密度で表現)
- **発展段階は地面に反映**: 土と草はら → 石畳 → 美しい敷石+花壇+水路
- 昼夜サイクルは盤面全体の `filter`(明るさ)と窓明かりの `opacity` で変化。夕方に窓へ一つずつ灯が入る演出も維持
- アニメーションは transform / opacity / filter のみ(毎フレームの layout / paint を起こさない)
- `window.LIZA_USE_DIORAMA = false` を `js/main.js` 読み込み前に設定すると、旧・横から見た夜景描画に戻せます(旧コードは温存)

## 実装済み機能

- オープニング「約束」(タップで進む4枚、スキップ可、「ずかん」から再生可)
- 昼夜サイクル(朝50秒/昼70秒/夕50秒/夜70秒。約30秒のCSS transitionでシームレスに変化。夕方は窓が時間差で順に点灯、昼は雲が流れる。位相はセーブされ続きから流れる)
- 願い星(45〜90秒に1回出現、約6秒で消える。夜=流れ星/昼=光る蝶。獲得は自動生産45秒ぶん・序盤は固定15)
- ものがたり図鑑(8ページ。施設Lv・発展段階の条件達成で解放、解放時トースト通知)
- タップによる光の獲得(光の粒・カウントアップ表示)
- 施設5種の「建てる/育てる」と、街への建物の出現・成長(セーブ復元でも同じ街並みを再現)
- 住人システム(合計Lv×2人。窓明かりが住人数と連動。節目では職業+一言つきの住人10人)
- 仲間の声(レイン=青/ランタン、バルト=橙/天秤、アルノ=白銀/羽根ペンの吹き出しトースト。各5種以上)
- 「はじめての建設」「はじめての住人」では必ず仲間が声をかける
- 発展段階5段階(段階=土地と空気、施設=建物、と役割分離)と光の波+「おめでとう!」祝祭演出
- 段階到達ストーリー(原文のまま・難読語のみルビ)と「図鑑」からの再読
- 「次の目標」チップ(状況から自動判定)
- 小学校高学年〜高校生向けの文体(通常の漢字仮名交じり・難読語のみルビ)
- localStorage自動セーブ(5秒ごと+操作時+画面非表示時)・確認つきリセット
- 数値の日本語略記(1.2万 など)、スマホ縦画面(390px基準)
- 「あたたかい案内板」UIテーマ(クリーム色の紙×こげ茶の文字、蜂蜜色のボタン、CSS自作アイコンつきフッター)
- Three.jsによる3Dミニチュア街ビュー(上記参照)+CSSジオラマ/旧描画への自動フォールバック
- 街のおねがい(納品/達成/二択の11種。装飾報酬5種が街に永続配置。二択は10分間の生産ブースト)

## セーブリセット方法

1. フッターメニューの「**設定**」をタップ
2. 「**セーブデータをリセット**」をタップ
3. 確認ダイアログで「OK」を選ぶと、最初から始まります

(手動で消す場合はブラウザの開発者ツールから localStorage の `toukaLizaSave_v1` を削除してください)

## 既知の課題

- バックグラウンドタブでは生産が最大300秒ぶんまでしか加算されません(暴走防止のための仕様)
- localStorageが使えない環境(プライベートブラウズ等)ではセーブされません(プレイ自体は可能)
- 横画面・タブレットでも動作はしますが、レイアウトは縦持ちスマホに最適化されています
- 効果音・BGMは仕様により未実装です
- 3DはWebGLを使用します。非対応環境ではCSSジオラマへ自動で切り替わります(さらに古い環境は旧描画へ)
- 仲間の紋章(🕯/⚖/✒)は端末の絵文字フォントに依存します(非常に古い端末では表示が変わる場合があります)
