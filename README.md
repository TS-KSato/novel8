# 灯火の街リーザ

スマートフォン専用のWEBブラウザゲーム（街づくり×放置クリッカー）です。
HTML / CSS / JavaScript のみで動作し、ビルド・サーバー・外部通信は一切不要です。

## ゲーム概要

魔力の量と血統がすべてを決める世界。落ちこぼれ扱いされた貴族の少年レインは、「観察し、記録し、工夫する」という知恵で魔法の常識を覆し、仲間たちと共に荒れ地に街を築いていきます。街の名は「リーザ市」——若くして亡くなった、かけがえのない友の名前。彼女が灯したたった一つのランタンの光を、街全体の光の海に変えていく物語です。

プレイヤーはレインたちと共にリーザ市を育てます。資源を集め、施設を建て、街が少しずつ明るく、にぎやかになっていく過程そのものがこのゲームの体験の核です。

## 遊び方

1. **画面上部の「街ビュー」をタップ**すると資源「魔素」が手に入ります（タップごとに+1〜。光の粒が舞い、街の灯が脈動します）
2. 魔素を消費して**施設を建設・強化**します。施設は自動で資源を生産し、**街ビューに対応する建物が実際に出現・成長**します（工房の窓灯り、市場の屋台とにぎわい、学問所の塔の青白い灯、救護院の優しい灯、街灯網の光点）
3. 一部の施設（市場）は第二の資源「**資材**」を生産します。上位施設の建設に必要です
4. 施設の合計レベルに応じて「**発展度**」が上がります
5. 発展度がしきい値を超えると「**発展段階**」が進み、土地と空気が変化し、ストーリーが解放されます
6. 迷ったら資源バー下の「**次の目標**」チップを見れば、次の一手が分かります

### 資源

| 資源 | 入手方法 | 用途 |
|---|---|---|
| 魔素 | タップ／ランタン工房の自動生産 | 全施設の建設・強化 |
| 資材 | 市場の自動生産 | 上位施設の建設・強化 |

### 施設（5種類）

| 施設名 | 担当 | 効果 | 初期コスト |
|---|---|---|---|
| ランタン工房 | レイン | 魔素の自動生産 +1/秒×Lv | 魔素15 |
| 市場 | バルト | 資材の自動生産 +1/2秒×Lv | 魔素40 |
| 学問所 | アルノ | タップ獲得量 +1×Lv | 魔素100・資材10 |
| 救護院 | リーザの遺志 | 全自動生産に +10%×Lv | 魔素250・資材30 |
| 街灯網 | ユリウス | 発展度 +5×Lv・街の光が増える | 魔素600・資材80 |

コストはレベルごとに約1.6倍ずつ増えていきます。

### 発展段階（5段階）

| 段階 | 名称 | 必要発展度 |
|---|---|---|
| 1 | 泥のアジト | 0 |
| 2 | 開拓の集落 | 5 |
| 3 | 灯りの街 | 14 |
| 4 | 交易都市 | 30 |
| 5 | 光の都リーザ | 55 |

段階が上がる瞬間には、光が波のように街へ広がる演出と共にストーリーメッセージが表示されます。一度見たメッセージはフッターの「記録」からいつでも再読できます。

## スマホ操作方法

- **街ビュー（上部）をタップ** … 魔素を獲得（連打OK）
- **施設カードの「建設／強化」ボタン** … 資源を消費して施設をレベルアップ
- **フッターメニュー**
  - 「街」「施設」… メイン画面（施設リストを先頭へスクロール）
  - 「記録」… 解放済みストーリーの再読
  - 「設定」… セーブ情報の確認・セーブデータのリセット
- 縦持ち・片手・親指操作前提のレイアウトです（横スクロールなし、ボタンは44px以上）

## GitHub Pages公開手順

1. このリポジトリの **Settings → Pages** を開く
2. **Source** で「Deploy from a branch」を選択
3. **Branch** で `main` / `/ (root)` を選択して **Save**
4. 数分待つと `https://<ユーザー名>.github.io/novel8/` で公開されます

※ ローカルで試す場合は `index.html` をブラウザで開くだけで動作します。

## 実装済み機能

- タップによる魔素獲得（光のエフェクト＋獲得量フロート＋光の粒パーティクル＋街の灯の脈動）
- 施設5種の建設・レベルアップ（コストはLvごとに約1.6倍で逓増）
- **施設が街の絵になる**: 建設・強化すると街ビューに対応する建物シルエットが出現・成長（Lvで窓灯り・屋台が増加。セーブ復元時も同じ街並みを再現）
- 建設・強化時の建物出現演出（光と共に成長）、建設可能になった瞬間のボタン明滅
- 自動生産（ランタン工房：魔素／市場：資材／救護院：全自動生産+10%×Lv）
- 学問所によるタップ獲得量の増加
- 発展度と5段階の発展段階（土地と空気＝背景がCSSのみで段階的に変化。地平線の薄明が段階ごとに明るくなる）
- 段階アップ時の「光が波のように広がる」演出
- 段階到達時のストーリーメッセージ（モーダル表示・「記録」から再読可能）
- 初回起動時の導入メッセージ（3行）
- 「次の目標」チップ（現在の状況から次の一手を自動判定して常時表示）
- 資源表示のカウントアップ演出、数値の日本語略記表示（1.2万 / 3.4億 など）
- localStorageへの自動セーブ（5秒ごと＋操作時＋画面非表示時）と自動ロード
- セーブデータのリセット（確認ダイアログ付き）
- スマホ縦画面（390px基準）レイアウト・「灯りが主役の夜」をテーマにした配色

## localStorage保存内容

キー: `toukaLizaSave_v1`

```json
{
  "v": 1,
  "maso": 123.4,
  "shizai": 56.7,
  "lv": { "lantern": 3, "market": 2, "school": 1, "clinic": 0, "lights": 0 },
  "unlocked": [2, 3],
  "maxStage": 3,
  "introSeen": true,
  "lastSaved": 1750000000000
}
```

| 項目 | 内容 |
|---|---|
| `maso` / `shizai` | 所持資源（内部では小数で保持、表示は整数） |
| `lv` | 各施設のレベル（街ビューの建物シルエットはここから決定的に再描画） |
| `unlocked` | 解放済みストーリーの段階番号 |
| `maxStage` | 到達済みの最高発展段階 |
| `introSeen` | 初回導入メッセージを表示済みか |
| `lastSaved` | 最終保存時刻（UNIXミリ秒） |

## セーブリセット方法

1. フッターメニューの「**設定**」をタップ
2. 「**セーブデータをリセット**」ボタンをタップ
3. 確認ダイアログで「OK」を選ぶと、データが削除されて最初から始まります

（手動で消す場合はブラウザの開発者ツールから localStorage の `toukaLizaSave_v1` を削除してください）

## 既知の課題

- バックグラウンドタブでは生産が最大300秒ぶんまでしか加算されません（暴走防止のための仕様）
- localStorageが使えない環境（プライベートブラウズ等）ではセーブされません（プレイ自体は可能）
- 横画面・タブレットでも動作はしますが、レイアウトは縦持ちスマホに最適化されています
- 効果音・BGMは仕様により未実装です
