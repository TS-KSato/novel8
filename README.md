# 灯火の街リーザ（仮）— カードパズル

人々の悩みを、手札のカードを組み合わせ、現代知識で解決するカードパズル。
プレイヤーはレイン。この版は**問1〜5を順番に解く試作**です。

## 遊び方

1. 画面の「悩み」を読む（ヘッダのタグに「悩み 1 / 5」と進み具合が出る）
2. 手札（火 / 水 / 鍋 / 布 / 灰 / 油 / 壺 / 風 ※問題ごとに変わる）から、解決に使うカードをタップで選ぶ（複数可・順不同）
   - もう一度タップで選択解除。「選び直す」で全解除
   - 選択中のカードは上の「レシピ」欄にも出る（チップのタップでも外せる）
3. 「これで解く」で確定 → 判定 → 解決後のテキストが出る
4. 結果に応じて**信頼**が増減（成功 +1 / 失敗 −1）。信頼が 0 未満でゲームオーバー
5. 成功なら「次へ」で次の問題へ。失敗なら同じ問題に再挑戦（信頼は持ち越し）
6. 問5を解くと終了表示。信頼は5問を通して持ち越します

## 動かし方

ビルド不要・サーバー不要。`index.html` をブラウザで直接開くだけ（スマホ縦 390px 基準）。

## ファイル構成

```
index.html       … 画面構造
css/style.css    … スタイル（手札の選択操作を最優先で作り込み）
js/data.js       … 問題・カードの静的データ（window.LisaCards）
js/main.js       … ゲームループ・判定・信頼ゲージ・保存
assets/portraits … 立ち絵・顔（共有資産。レインの顔のみ使用）
```

## 保存

`localStorage` のみ。キーは接頭辞 `lisa_cards:` 付き（信頼は `lisa_cards:trust`、進捗は `lisa_cards:progress`）。
旧データと混ざらないよう、このゲームのキーはすべてこの接頭辞を使います。

## この版の範囲

問1〜5を順番に解く流れまで。判定・選択UI・信頼ゲージは問1の縦スライスをそのまま再利用しています。「解けた瞬間の演出」（Three.js / GLB）はこの版では読み込みません（次フェーズ）。
