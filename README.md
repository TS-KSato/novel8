# 灯火の街リーザ（仮）— カードパズル（v2）

人々の悩みを、手札のカードを組み合わせ、現代知識で解決するカードパズル。
プレイヤーはレイン。この版は**問1〜5を順番に解く試作**です。

## 遊び方

1. 画面の「悩み」と、その下の**観察メモ**（レインの覚え書き・無料・常時表示）を読む（ヘッダのタグに「悩み 1 / 5」と進み具合が出る）
2. 手札（火 / 水 / 鍋 / 布 / 風 / 土 / 壺 / 油 / 木 ※問題ごとに変わる）から、解決に使うカードをタップで選ぶ（複数可・順不同）
   - もう一度タップで選択解除。「選び直す」で全解除
   - 選択中のカードは上の「レシピ」欄にも出る（チップのタップでも外せる）
3. 詰まったら**仲間に相談**できる（浅いヒント→深いヒントの二段）。または**相談を断って**未解決のまま次へ送る
4. 「これで解く」で確定 → 判定 → 解決後のテキストが出る
5. 結果に応じて**信頼**が増減。成功なら「次へ」、失敗なら同じ問題に再挑戦
6. 問5まで（解決 or 断る）埋まると終了表示

## 信頼ゲージ（増減値は `js/main.js` の `DELTA` に一元化・暫定）

| 操作 | 信頼 |
| --- | --- |
| 成功 | +2 |
| 失敗 | −2 |
| 深ヒント | −2 |
| 浅ヒント | −1 |
| 相談を断る | −1 |
| 観察メモ | 0（無料） |

- スタート 0。**0未満でゲームオーバー**。
- ヒントと「断る」は**消費後も信頼が0以上に保てるときだけ**選べる（足りないときはボタン無効＋「信頼が足りない」表示）。これにより、実際にゲームオーバーを起こすのは「失敗」だけ。信頼が無いときは相談も撤退もできず「解くしかない」緊張になる。
- ゲームオーバー → 「この問題からやり直す」。**問1へは戻らず、今いる問題の頭から再開**（信頼0に戻す）。解いた問題は解答済みとして保持。

## 動かし方

ビルド不要・サーバー不要。`index.html` をブラウザで直接開くだけ（スマホ縦 390px 基準）。

## ファイル構成

```
index.html       … 画面構造（観察メモ・相談/撤退ボタンを追加）
css/style.css    … スタイル（手札の選択操作を最優先で作り込み）
js/data.js       … 問題・カードの静的データ（window.LisaCards）
js/main.js       … ゲームループ・判定・信頼ゲージ・相談/撤退・保存
assets/portraits … 立ち絵・顔（共有資産。レインの顔のみ使用）
```

## 保存

`localStorage` のみ。キーは接頭辞 `lisa_cards:` 付き（信頼 `lisa_cards:trust`、進捗 `lisa_cards:progress`、各問の状態 `lisa_cards:status`）。
旧データと混ざらないよう、このゲームのキーはすべてこの接頭辞を使います。

## この版の範囲

問1〜5を順番に解く流れ＋観察メモ・相談ヒント（浅/深）・撤退・ゲームオーバー後の再開位置まで。判定・選択UI・信頼ゲージは既存のものをそのまま再利用しています。「解けた瞬間の演出」（Three.js / GLB）/ オープニング画面 / 章構造はこの版では入れていません（次フェーズ）。
